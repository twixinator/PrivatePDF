## ‚ö†Ô∏è Important

The font files are **not included in the repository** due to size considerations. You must download them manually before running the application.

**Quick Start:**
1. Download Cormorant: https://fonts.google.com/specimen/Cormorant
2. Download Inter: https://fonts.google.com/specimen/Inter
3. Place the `.ttf` files in this directory
4. Run `flutter pub get`
5. Run `flutter run -d chrome`

Both fonts are **free and open source** (SIL OFL 1.1 License).
